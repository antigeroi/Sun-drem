# Sunny Dream Bot 🌞

Полноценный RPG-бот для фэнтези ролевого сервера. Весь мир строится на РП-взаимодействии игроков. Бот дает инструменты для атмосферы, экономики, политики и фиксации событий, но не заменяет РП механиками.

## ✨ Основные возможности

### 👤 Персонажи
- Создание персонажа с уникальным префиксом
- Автоматическое получение случайного рода
- Биография (заполняется 1 раз, нельзя изменить)
- Паспорт (выпадает при смерти, подбирает убийца)

### 🎭 Ролевые команды
- `/шепот` - личное сообщение в общем чате
- `/мысль` - показать мысли персонажа
- `/эмоция` - описать действие
- `/осмотреться` - описание локации

### 💰 Экономика
- Валюта - астралы
- Передача денег и предметов
- Продажа предметов игрокам
- Магазины (только локальные)

### ⚒️ Крафт и книги
- Рецепты крафта (базовые и книжные)
- Книги крафта падают с 20% шансом
- 10% шанс неудачи при крафте

### 🏰 Гильдии
- Создание за 100 астралов
- Общий банк (деньги + предметы)
- Налог 5% при снятии

### 🎯 Охота за головами
- Назначить награду за игрока
- 3 дня на выполнение
- Автоматическое зачисление при смерти цели

### 💍 Браки
- Предложение, помолвка, свадьба
- Семейный банк
- Наследование при смерти

### ⚔️ Дуэли
- Вызов на дуэль со ставкой
- Минимум 2 свидетеля
- Проигравший умирает

### 🗺️ Путешествия
- Реальное время в пути (настраивается админом)
- Канал дороги для всех путников
- 30% шанс встретить другого путника
- Случайные атмосферные события

### 👑 Титулы
- Титулы с зарплатой
- Наследование титулов
- Титулы с условиями
- Система разрешений на выдачу

### 📮 Письма
- Физические письма (предметы)
- Перехват (30% шанс)
- Подделка (70% шанс)
- Запечатывание и шифрование

### 🏛️ Хранилища и налоги
- Создание хранилищ (админ)
- Доступ по ролям
- Налоги с покупок
- Статистика за 3 часа

### 🤖 NPC
- Горожане, путники, жители
- Случайные встречи
- Убийство NPC (возрождаются)
- Свидетели убийств (50% шанс)
